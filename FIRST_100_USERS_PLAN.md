## ONBOARDING USERS

### First User Experience:

**Day -3 to -1: Pre-invite**

- Send personalized invite with app link
- Set expectation: "Will you try and give feedback?"

**Day 0: Download + Signup**

- Track: Did they download? Signup?
- Action: Follow up within 24h if no

**Day 1-3: Create First Offer/Request**

- Personal check-in: "How's it going?"
- Help if stuck: Send template offer/request
- Action: Nudge if no first action by Day 3

**Day 7: First Week Check**

- Ask: "What would make this better?"
- Collect feedback
- Fix issues

## ACTIVATING USERS

### Activation Goals:

| Milestone           | Target Time | How                         |
| ------------------- | ----------- | --------------------------- |
| Signup              | Immediate   | Direct link                 |
| Onboarding complete | <48h        | AI nudges                   |
| First offer/request | <7 days     | Personal follow + templates |
| First proposal      | <14 days    | AI match                    |
| First deal          | <30 days    | Full experience             |

### Activation Flow by User Type:

#### Provider Activation:

```
1. Signup
2. Onboarding
3. Create offer (with AI price help)
4. Post offer to search
5. Receive proposals
6. Accept proposal / Deal
```

#### Requester Activation:

```
1. Signup
2. Onboarding
3. Create request (with AI price help)
4. Receive proposals
5. Accept proposal / Deal
```

### Activation Support:

| For                    | Do This             | When  |
| ---------------------- | ------------------- | ----- |
| Provider no offer      | Send template offer | Day 3 |
| Provider no views      | Offer SEO tips      | Day 5 |
| Requester no proposals | Review pricing      | Day 5 |
| Any stuck              | Personal help       | Day 7 |
