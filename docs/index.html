Public Access (Codespaces)

Để cho khách hàng truy cập từ bên ngoài, bạn có thể xuất bản cổng từ Codespaces. Mục tiêu là có một URL public để gọi API backend (ví dụ health/offers) từ Internet.

  1. Trong Codespace, mở tab Ports. Chọn cổng 3000 và bật Public (Expose public port).
  2. Trình duyệt sẽ hiển thị một URL public như: https://-3000.preview.app.github.dev/
  3. Kiểm tra endpoint, ví dụ health: https://-3000.preview.app.github.dev/api/v1/health
  4. Lưu ý: URL công khai này là tạm thời và phụ thuộc trạng thái codespace. Có thể mất khi codespace dừng hoặc bị xóa.

Nếu không có quyền expose port công khai từ Codespaces, dùng ngrok làm giải pháp tạm thời (tài liệu ngắn gọn bên dưới):

ngrok http 3000

Ngrok sẽ trả về một URL công khai, ví dụ https://abcd1234.ngrok.io, bạn có thể dùng cho các thử nghiệm nhanh.
