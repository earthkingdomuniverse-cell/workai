# WorkAI Post-Launch 30-Day Plan

## Purpose

First-month execution plan after WorkAI 1.0 launch.

## Week 1: Stabilization

Primary focus: stabilize launch, fix critical bugs, and get initial users activated.

### Metrics to watch

- Signups
- Active offers
- Active requests
- First actions
- App crashes
- API errors

### Likely fires

- Crash on login
- Offer create fails
- AI match fails
- Signup volume is low

### Key actions

1. Monitor metrics daily.
2. Respond to user feedback within 24 hours.
3. Fix crash or core-flow bugs immediately.
4. Follow up with inactive users.
5. Track onboarding completion.

## Week 2: Activation

Primary focus: move users to first value, especially first offer or first request.

### Metrics to watch

- Signups
- Active offers
- Active requests
- First offers created
- First requests created
- Proposals
- Onboarding completion

### Key actions

1. Follow up with users who signed up but did not activate.
2. Create or improve templates.
3. Analyze drop-off points.
4. Improve onboarding nudges.

## Week 3: Engagement

Primary focus: get users to first proposal and first deal.

### Metrics to watch

- Proposals sent
- Deals created
- Deals funded
- D7 retention
- First reviews

### Key actions

1. Manually match where AI or liquidity is weak.
2. Follow up on deals stuck in funding.
3. Encourage reviews after completed deals.
4. Track retention closely.

## Week 4: Scaling

Primary focus: scale user acquisition and prepare 1.1 priorities.

### Metrics to watch

- Signups
- Active offers
- Active requests
- Deals created
- Deals completed
- D30 retention
- Weekly growth

### Key actions

1. Test referral and acquisition loops.
2. Review user feedback.
3. Prioritize the 1.1 feature list.
4. Review dispute, review, and satisfaction trends.

## 30-day targets

| Goal | Target |
| --- | --- |
| Signups | 500 |
| Active offers | 100 |
| Active requests | 50 |
| Deals completed | 15 |
| Reviews | 10 |
| D30 retention | Above 20% |

## Weekly questions

- Are we growing?
- Are users activating?
- Are deals happening?
- Are we fixing issues fast?
- Is the operating load sustainable?
