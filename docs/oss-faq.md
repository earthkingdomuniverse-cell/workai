# OSS FAQ

- Q: What is the licensing for WorkAI core?
- A: We adopt an MIT-style license for the Core. See LICENSE for details.
- Q: How can I contribute?
- A: Read CONTRIBUTING.md. Open issues with feature requests or submit PRs with small, well-scoped changes.
- Q: How do plugins work?
- A: Plugins live under plugins/ and are loaded via the Open Source loader (src/open_source/index.ts). Implement a plugin with a minimal interface and submit as a PR.
- Q: How are monetization and open core balanced?
- A: The Open Core pattern can be used: core features are free; advanced enterprise features or hosted services are offered under paid licensing or SLA plans.
