# Code of Conduct

This project adheres to the Contributor Covenant Code of Conduct. By contributing, you agree to abide by this code.

Core principles:

- Be respectful and inclusive.
- Be mindful of the diverse backgrounds of contributors.
- Refrain from harassment, hate speech, or intimidation.
- Use welcoming and constructive language in all interactions.
- Respect others’ privacy and personal information.
- Report misconduct via the SECURITY.md flow or the repository's issue form.

Enforcement:

- Maintainers may take action on any behavior that disrupts the community, including warning, temporary bans, or permanent bans, at their discretion.

Scope:

- This code applies to maintainers, contributors, and participants in all repository-related discussions, issues, PRs, and community events.
