## KIẾM TIỀN TỪ ĐÂU

| Revenue Stream           | Description        | Priority  |
| ------------------------ | ------------------ | --------- |
| Platform fee (take rate) | % of deal value    | PHASE 1.2 |
| Premium visibility       | Featured placement | PHASE 1.1 |
| Pro accounts             | Enhanced features  | PHASE 1.x |

---

## KHÔNG KIẾM TIỀN TỪ ĐÂU

| What NOT                | Reason                |
| ----------------------- | --------------------- |
| User signups            | Growth barrier        |
| Basic profile           | Trust foundation      |
| Viewing offers/requests | Marketplace liquidity |
| Messaging (when ready)  | Engagement            |
| Reviews                 | Trust foundation      |

---

## THỨ TỰ ƯU TIÊN

1. **Phase 1.0**: NO monetization (free for growth)
2. **Phase 1.1**: Premium visibility (minimal)
3. **Phase 1.2**: Platform fee (5-10% take rate)
4. **Phase 1.x**: Pro accounts, enterprise

---

## NGUYÊN TẮC

### Không Phá Trust

- Never charge for reviews
- Never sell trust scores
- Trust must remain authentic

### Không Phá Conversion

- Never block core actions behind paywall
- Provider must be able to get first deal without paying
- Requester must be able to create request without paying

### Không Bóc Lột User Mới

- Never charge for first action
- Never charge for onboarding
- Never charge for profile creation

---

## CORE PRINCIPLE

> **Monetization serves marketplace health, not extraction.**

Platform fee should improve quality, not gate access.
