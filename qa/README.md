# WorkAI QA Hub

This folder contains the quality-assurance documentation for WorkAI.

## Documents

- `test-plan.md` - end-to-end QA strategy
- `backend-checklist.md` - backend API, database, wallet, payment, and deployment checks
- `mobile-checklist.md` - mobile app QA checks
- `payment-wallet-checklist.md` - wallet, ledger, escrow, deposit, release, withdrawal, and provider callback checks
- `deployment-checklist.md` - local, staging, and production deployment verification
- `bug-report-template.md` - standard bug report template

## QA Priority

1. Money correctness
2. Wallet and ledger consistency
3. Payment callback idempotency
4. Escrow hold/release accuracy
5. Withdrawal failure safety
6. Auth and permission boundaries
7. Mobile payment/deep-link flow
8. Deployment reproducibility
