# Chính sách bảo mật (Security Policy)

## Các phiên bản được hỗ trợ

Hiện tại chúng tôi chỉ hỗ trợ bảo mật cho phiên bản mới nhất trên branch `main`.

| Phiên bản | Hỗ trợ |
| --------- | ------ |
| 1.0.x     | ✅     |
| < 1.0     | ❌     |

## Báo cáo lỗ hổng

Nếu bạn phát hiện lỗ hổng bảo mật, vui lòng **không** báo cáo công khai qua GitHub Issues.

Ưu tiên dùng `Private vulnerability reporting` trong tab `Security` của repository để gửi báo cáo riêng tư cho maintainers. Nếu repo chưa bật tính năng này, hãy dùng một kênh liên hệ riêng tư với maintainer và đính kèm:

- Mô tả lỗ hổng.
- Bước tái hiện.
- Phạm vi ảnh hưởng.
- Đề xuất cách giảm thiểu nếu có.

Mục tiêu phản hồi ban đầu là trong vòng 48 giờ.
