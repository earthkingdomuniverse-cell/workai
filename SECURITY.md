# Chính sách bảo mật (Security Policy)

## Các phiên bản được hỗ trợ

Hiện tại chúng tôi chỉ hỗ trợ bảo mật cho phiên bản mới nhất trên branch `main`.

| Phiên bản | Hỗ trợ |
| --------- | ------ |
| 1.0.x     | ✅     |
| < 1.0     | ❌     |

## Báo cáo lỗ hổng

Nếu bạn phát hiện lỗ hổng bảo mật, vui lòng **không** báo cáo công khai qua GitHub Issues. Thay vào đó, hãy gửi email trực tiếp cho đội ngũ quản trị.

Chúng tôi sẽ phản hồi trong vòng 48 giờ và cung cấp lộ trình khắc phục.
